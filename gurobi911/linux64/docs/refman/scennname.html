ScenNName

Type: string
Modifiable: Yes

When the model has multiple scenarios, this attribute is used to query or modify the name for scenario . You set using the ScenarioNumber parameter.

The number of scenarios in the model can be queried (or modified) using the NumScenarios attribute.

Please refer to the Multiple Scenarios discussion for more information.

For examples of how to query or modify attributes, refer to our Attribute Examples.
