MLinExpr.__ge__()

__ge__ ( arg )

Overloads the >= operator, creating a TempConstr object that captures a 1-D array of inequality constraints. The result is typically immediately passed to Model.addConstr.

Return value:

A TempConstr object.

Example usage:

  m.addConstr(A @ x >= 1)
