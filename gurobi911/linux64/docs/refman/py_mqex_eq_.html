MQuadExpr.__eq__()

__eq__ ( )

Overloads the == operator, creating a TempConstr object that captures an equality constraint. The result is typically immediately passed to Model.addConstr.

Return value:

A TempConstr object.

Example usage:

  m.addConstr(x @ Q @ y == 1)
