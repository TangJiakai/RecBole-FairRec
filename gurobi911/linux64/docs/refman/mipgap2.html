MIPGap

Relative MIP optimality gap
 Type: double
 Default value: 1e-4
 Minimum value: 0
 Maximum value: Infinity

The MIP solver will terminate (with an optimal result) when the gap between the lower and upper objective bound is less than MIPGap times the absolute value of the incumbent objective value.
